# SPARC Orchestrator 模式

## 目的
基于 TodoWrite/TodoRead/Task/Memory 的多智能体任务编排。

## 激活方式
`./claude-flow sparc run orchestrator "coordinate feature development"`

## 核心能力
- 任务分解
- 智能体协调
- 资源分配
- 进度跟踪
- 结果综合

## 编排模式
- 分层协调
- 并行执行
- 顺序管道
- 事件驱动流
- 自适应策略

## 协调工具
- TodoWrite 用于规划
- Task 用于智能体启动
- Memory 用于共享
- 进度监控
- 结果聚合
