# Analysis Swarm 策略

## 目的
通过分布式智能体协调进行全面分析。

## 激活方式
`./claude-flow swarm "analyze system performance" --strategy analysis`

## 智能体角色
- Data Collector: 收集指标和日志
- Pattern Analyzer: 识别趋势和异常
- Report Generator: 创建综合报告
- Insight Synthesizer: 整合研究结果

## 协调模式
- Mesh: 用于探索性分析
- Pipeline: 用于顺序处理
- Hierarchical: 用于复杂系统
