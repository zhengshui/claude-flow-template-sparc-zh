# Development Swarm 策略

## 目的
通过专业智能体团队进行协调开发。

## 激活方式
`./claude-flow swarm "build feature X" --strategy development`

## 智能体角色
- Architect: 设计系统结构
- Frontend Developer: 实现用户界面
- Backend Developer: 创建 API
- Database Specialist: 管理数据层
- Integration Expert: 连接组件

## 最佳实践
- 对大型项目使用分层模式
- 启用并行执行
- 实施持续测试
